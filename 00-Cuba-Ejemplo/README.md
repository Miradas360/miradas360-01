## ¿Cómo ejecutar el proyecto localmente?
 
###Instalar NodeJS
[Node Js Oficial](http://nodejs.org)


###Instalar http-server
`npm install http-server -g`

###Ir a Directorio DIST y ejecutar servidor

`cd dist`

`http-server ./`

###Abrir navegador 

Dirección: `http://localhost:8080`
